# Contributing to Movie Recommendation System

Thank you for your interest in contributing to this project! We appreciate your time and effort. Here are a few guidelines to follow:

## Getting Started

1. Fork the repository
2. Clone your fork locally:
   ```bash
   git clone https://github.com/yourusername/movie-recommendation-system.git
   ```
3. Create a new branch for your feature:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. Install the development dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Making Changes

1. Make your changes to the code
2. Test your changes thoroughly
3. Add comments to explain any new functionality
4. Follow the existing code style and conventions

## Submitting Changes

1. Commit your changes with a clear, descriptive message:
   ```bash
   git commit -m "Add feature: description of your changes"
   ```
2. Push to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
3. Submit a pull request to the main repository

## Code Style

- Follow PEP 8 style guidelines
- Use meaningful variable and function names
- Add docstrings to functions
- Include comments for complex logic

## Reporting Issues

If you find a bug or have a suggestion:

1. Check if the issue already exists
2. Provide a clear description of the issue
3. Include steps to reproduce the problem
4. Explain the expected vs. actual behavior

## Questions?

Feel free to open an issue for any questions or discussions.

Thank you for contributing!
